# BD商机管理系统 - 使用说明

## 项目概述

BD商机管理系统是一个用于管理公司BD重要事项（商机）跟进的前端应用，支持Web端和H5端（企微）访问。

## 功能模块

### 1. 商机列表

**功能说明：**
- 支持多维度筛选：事项搜索、年度、创建时间、客户、跟进人等
- 支持多种排序方式：按计划完成时间、创建时间、最后更新时间
- 显示商机的完整信息，包括状态、重要程度、进展等
- 支持查看详情和编辑操作

**使用方法：**
1. 进入"商机列表"页面
2. 使用筛选条件筛选需要的商机
3. 选择排序方式对列表进行排序
4. 点击"详情"查看商机详细信息
5. 点击"编辑"修改商机信息

### 2. 客户管理

**功能说明：**
- 客户列表查看，支持搜索和KA客户筛选
- 客户详情查看
- 客户新增和编辑，支持客户去重校验
- 支持关联PMS客户

**使用方法：**
1. 进入"客户管理"页面
2. 可以搜索客户名称或筛选KA客户
3. 点击"新增客户"创建新客户（系统会自动检查重复）
4. 点击"详情"查看客户信息
5. 点击"编辑"修改客户信息

### 3. 看板视图

#### 上周新增商机看板
- 按跟进人分组展示上周新增的商机
- 卡片形式展示，按创建时间和状态排序
- 点击卡片可查看商机详情

#### KA客户事项看板
- 按KA客户分组展示所有相关商机
- 卡片形式展示，便于快速查看

### 4. 权限管理

**功能说明：**
- 支持为不同角色配置字段可见性和可编辑性
- 支持配置字段的预选项
- 支持设置字段是否必填

**使用方法：**
1. 进入"权限管理"页面
2. 点击"配置字段"按钮
3. 在弹窗中配置每个字段的可见性、可编辑性等属性
4. 保存配置

### 5. 提醒设置

**功能说明：**
- 支持配置提醒规则
- 可以设置触发日期（每周几）
- 可以设置提前天数
- 可以自定义提醒内容
- 支持启用/禁用提醒规则

**使用方法：**
1. 进入"提醒设置"页面
2. 点击"新增提醒规则"
3. 配置规则名称、触发日期、提前天数、提醒内容
4. 保存规则

## 平台支持

### Web端
- 完整的后台管理功能
- 侧边栏导航
- 响应式布局，适配不同屏幕尺寸

### H5端（企微）
- 适配移动端操作
- 抽屉式导航菜单
- 触摸友好的交互设计
- 支持在企微聊天框中打开H5页面

## 开发指南

### 环境要求
- Node.js >= 16.0.0
- npm >= 8.0.0

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

### 代码规范
- 使用ESLint进行代码检查
- 使用TypeScript进行类型检查
- 遵循React Hooks最佳实践

## API接口说明

所有API接口统一使用 `/api` 作为基础路径，具体接口文档需要后端提供。

### 接口规范
- 请求格式：JSON
- 响应格式：`{ code: 200, data: {...}, message: "..." }` 或 `{ success: true, data: {...} }`
- 认证方式：Bearer Token（存储在localStorage中）

## 注意事项

1. **数据权限**：系统支持指定业务人员与管理层可见和编辑数据
2. **客户去重**：创建客户时系统会自动检查客户名称是否已存在
3. **跟进提醒**：提醒功能需要后端配合实现定时任务
4. **企微集成**：H5端需要配置企微应用入口和消息推送功能
